## Iris Prediction & Analytics Platform
An end-to-end machine learning web application for classifying Iris flower species, with a real-time prediction API, database storage, and an interactive dashboard.

## Features
- Predict Iris flower species in real time
- Store prediction history in a database
- View predictions through an interactive dashboard

## Techniques Used
- Machine Learning Classification (Scikit-Learn)
- REST API Development
- Full-Stack Integration (Frontend + Backend + Database)

## Tools
- Scikit-Learn
- Flask
- React
- PostgreSQL
- Supabase

## How it works
The Flask API receives input data and uses a trained Scikit-Learn model to predict the Iris species. Each prediction is stored in a PostgreSQL database, and the React dashboard displays results and prediction history in real time.
